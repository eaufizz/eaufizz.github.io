各データの説明
①セット勝率
セット当たりの勝率です。
計算式：総勝利数/総プレイ数100
②勝利ターン平均
勝利時のターン数の平均値です。
この値が小さいほど最短で勝利できていることを示します。
計算式：勝利時ターン数の総和/総勝利数
③ブレイク平均
ブレイクにおける獲得点の平均値です。
計算式：ブレイク獲得点の総和/総ブレイク投擲数
④狙い通り率
狙い通りに投擲する確率です。
狙った通りの正確な投擲ができているかを示します。
計算式：総狙い通り数/総投擲数
⑤平均得点
一投当たりの獲得の平均値です。
戦略的に小さい点数を狙うこともありますが、強いプレイヤーであるほど点数の高いスキットルを最短で倒す結果、平均値が高くなります。
計算式：総獲得点/総投擲数
⑥ファウル率
点数を獲得できなかった投擲（ファウル）の確率です。
計算式：スキットルを倒せなかった回数の総和/総投擲数100
⑦50オーバー
セットあたりに50点をオーバーする回数（確率）を示します。チームのオーバーもカウントされます。
計算式：50点オーバーした回数の総和/総プレイセット数
⑧セット失格率
セットあたりに3回連続のファウルにより失格になる確率を示します。チームの失格もカウントされます。
計算式：失格した回数の総和/総プレイセット数*100
